# Growth Agent

个人微信小号推广草稿助手，用于微信小程序冷启动推广。

它做三件事：

- 管理推广活动、白名单群/好友/朋友圈目标。
- 自动生成带 `source` 来源码的小程序推广草稿。
- 记录草稿状态：待处理、已复制、已发送、已跳过。

它不做这些事：

- 不 hook 微信客户端。
- 不绕过验证码或风控。
- 不自动点击发送。
- 不批量骚扰陌生人。

## 运行

```powershell
cd E:\growth-agent
npm start
```

打开：

```text
http://localhost:4788
```

默认数据存储在：

```text
E:\growth-agent\data\growth-agent.sqlite
```

## 流程

1. 创建推广活动，例如“图片压缩冷启动”。
2. 批量添加白名单目标，例如“朋友圈”或多个微信群。
3. 点击“生成草稿”。
4. 在草稿队列点击“复制”。
5. 粘贴到 Windows 微信，人工确认后发送。
6. 回到后台点击“标记发送”。

批量添加白名单时，可以一行一个：

```text
A青岛装修合作共赢群9
A南京工程装修资源整合群6
A深圳装修设计交流群
```

也可以粘贴用逗号、分号分隔的列表。对于 `A... A...` 这种连续群名，系统会尝试按空格前的 `A` 自动拆分。添加错了，在“白名单目标”里点“删除”。

## 测试

```powershell
npm test
```

测试覆盖：

- 来源码和小程序路径生成。
- 推广草稿生成。
- 白名单、重复目标、每日上限拦截。
- SQLite 存储。
- 本地 API。

## 安全边界

第一版只做“草稿自动化”。后续如果接 Windows 微信自动化，也应保持“写入草稿，不自动发送”的边界。
